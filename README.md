# Ferrofauna: Wildcore Frontier

Premium offline-first sci-fi creature-collector RPG, built in Godot 4.x.

**This repository hosts released game builds.** Ferrofauna: Wildcore Frontier is **not**
open source — the source code is kept in a private repository. Packaged release builds
will be published here when they are ready.

No public build, early access, or paid release before the Final Gold Master.
